# UniRank Flutter App

A neon-themed Flutter UI for UniRank.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```

2.  **Run the App:**
    ```bash
    flutter run
    ```

## Project Structure

*   `lib/main.dart`: App entry point and routes.
*   `lib/theme.dart`: Neon theme configuration.
*   `lib/data/sample_data.dart`: Dummy data for UI.
*   `lib/screens/`: App screens (Feed, Dashboard, Profile, Heatmap).
*   `lib/widgets/`: Reusable neon widgets.

## Features

*   **Neon Theme:** Dark background with neon purple, blue, and green accents.
*   **Glassmorphism:** Frosted glass effects on cards and navigation.
*   **Responsive Layout:** Adapts to different screen sizes.
*   **Interactive Widgets:** Neon buttons, avatars, and heatmaps.
