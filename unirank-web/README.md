ğŸ§® The "Secret Sauce" Algorithm

We calculate the Employability Index (EI) using a weighted formula:

EI = (A \times 0.4) + (P \times 0.4) + (C \times 0.2)


A (Attendance): Consistency in classes.

P (Projects): GitHub activity and project complexity.

C (Certifications): Verified skills and badges.

This score normalizes to 1000, giving recruiters a single metric to gauge potential.
